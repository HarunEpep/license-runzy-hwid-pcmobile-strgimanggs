Cara Menyalin HWID

  • Masuk ke dalam Game GTA SAMP Anda.
  • Aktifkan script menggunakan perintah yang tertera di monetloader.
  • Jika lisensi belum aktif, klik tombol "SALIN ID" pada layar utama UI MonetLoader.
  • Tempel (Paste) kode tersebut ke kolom pencarian di atas.
